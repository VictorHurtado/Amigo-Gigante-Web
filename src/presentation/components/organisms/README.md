Organisms: componentes complejos construidos a partir de atoms y molecules.

- Resuelven bloques completos de UI (navbars, cards, secciones).
- Mantienen la responsabilidad de presentación, sin lógica de negocio.
- Reutilizan los atoms/molecules existentes para evitar estilos duplicados.
- Se documentan en cada carpeta con sus props principales si aplica.

### Organisms disponibles

- `HomeNavBar`: barra superior que usa `Logo`, `NavLink`, `SearchButton` y `Button`.
- `AnimalCard`: tarjeta de animal que aprovecha `StatusChip` e `IconButton` para estados y acciones.
